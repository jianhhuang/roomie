The team used htmlcodex.com template as front-end design please refer to LICENSE.txt

To run the website please do the following:
Download XAMPP and start both Apache and MySQL modules (Config Apache moudle and set LISTEN to 8080 or other port numbers)
Import roomie_db to phpMyAdmin to create the database and insert database
Once the database and server is ready please use this as a reference http://localhost:8080/*directory name*/index.php in the browser 
Now the server should be running
